# AI-OPERATORS - AI Solution Group

Your intelligent partner for enterprise AI automation.

## Features

- **WhatsApp AI Agent**: 24/7 customer support and sales automation.
- **Multilingual**: Full support for English and Arabic (RTL).
- **Modern UI**: Built with Next.js, Tailwind CSS, and Framer Motion.
- **Contact Integration**: Connected to Gmail for real-time inquiries.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
